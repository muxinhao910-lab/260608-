# Project Memory

## Project Identity

This is a personal website project for A股、港股、ETF、美股产业链信息查询.

It is not a normal K-line stock website.

The website should become an artistic, high-impact, 3D-feeling, scroll-driven industrial-chain research entrance.

## User Role

The user is the project owner and idea creator.

The user is not required to speak like a professional product manager or programmer.

Agents should translate the user's informal ideas into clear product requirements, coding tasks, and acceptance criteria.

## Core Product Direction

The website should help users understand long-term industrial-chain information, including:

- company customers
- orders
- revenue exposure
- gross margin changes
- capacity expansion
- supply chain relationships
- Taiwan hardware monthly revenue
- memory chip price trends
- ETF share changes
- macro and credit turning points
- industrial high-frequency signals

## Current Priority

1. Functional routes must work.
2. Sector pages must work.
3. Developer Mode must enter a real Builder.
4. Builder must allow adding, selecting, editing, saving, refreshing, and resetting website components.
5. Advanced homepage visual effects and scroll animations come later.

## Important Product Rule

A feature is not complete just because a page opens.

A feature is complete only when the user can reach it, click it, use it, and verify the result.

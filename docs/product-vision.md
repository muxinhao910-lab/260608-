# Product Vision

## Project Positioning

This is a personal website for A股、港股、ETF、美股产业链信息查询.

It is not a K-line chart website.

The website should help users understand what long-term industrial-chain information matters for companies, sectors, ETFs, and market themes.

## Core Content Direction

The website should focus on information such as:

- company customers
- orders
- revenue exposure by theme
- gross margin changes
- capacity expansion
- supply chain relationships
- Taiwan hardware monthly revenue
- memory chip price trends
- ETF share changes
- macro and credit turning points
- industrial high-frequency indicators

## Design Direction

The homepage should eventually feel like:

- an artistic industrial-chain research entrance
- high-end
- visually impactful
- 3D-like
- scroll-driven
- closer to an interactive digital artwork than a normal finance portal

## Current Priority

Do not prioritize animation recovery yet.

Current priorities:

1. Basic routes must work.
2. Sector pages must work.
3. Developer Mode / Builder must become real and usable.
4. Builder should allow the user to edit website components.
5. Advanced visual effects come later.

## Product Principle

Functional correctness comes before visual polish.

A page is not considered complete just because it renders.
A feature is not considered complete unless the user can reach it, click it, and see the expected result.
